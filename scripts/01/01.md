## Databases

- Accessing
    - SSH into your virtual server:
        `$ vagrant ssh`
    - access MySQL with the root user account
        `$ mysql -u root -p`
    - Password is `root`
    - Or, you can type:
        `$ mysql -u root -proot`

## SQL conventions

- "Special" words written in uppercase
    ```sql
    USE DATABASE ...;
    SELECT * FROM ...;
    CREATE TABLE ...;
    ```
- Quotes and backticks
    ```sql
    `table name`
    `field name`
    'string value'
    "string value"
   ```
